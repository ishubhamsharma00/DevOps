Types of k8s cluster setup:
- KIND (Kubernetes in Docker): single instance cluster
- Minikube: single instance cluster
- KubeADM
- EKS/AKS/GKE

Steps to setup a KIND cluster:
- Install KIND (https://kind.sigs.k8s.io/docs/user/quick-start/)
- Install Kubectl (https://kubernetes.io/docs/tasks/tools/)
- Install Docker
